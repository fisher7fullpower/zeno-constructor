# Project Profile — Morrow Lab

## Что это

AI-платформа дизайна интерьеров. Беларусь, Минск. `morrowlab.by`

Пользователь загружает фото комнаты → выбирает стиль → получает AI-рендер.
Дополнительно: заявки на ремонт, маркетплейс партнёров-дизайнеров, AI-консультант.

## Аудитория

- Дизайнеры и студии: партнёры платформы
- B2B: застройщики, строительные компании

## Архитектурные решения (не менять без обсуждения)

| Решение | Причина |
|---------|---------|
| Single-file HTML (нет сборщика) | Простой деплой через scp, нет CI/CD |
| JSON storage для заявок | Нет ORM, простое резервирование |
| «Ссылка = доступ» (URL токены) | Простота для пользователей без аккаунтов |
| Email через Resend (не SMTP) | Надёжная доставка, нет своего MTP |
| Groq вместо OpenAI | Стоимость, скорость, беларусь-friendly |
| n8n для workflow | Визуальная логика, без кода |
